# Requirements
Conda - make sure it is installed!
 conda create --name LyricSynth tensorflow-gpu==1.15.2
 or
 conda create --name LyricSynth tensorflow==1.15.2

 To enter the virtual environment:
  conda activate LyricSynth
 And then to exit:
  conda deactivate
